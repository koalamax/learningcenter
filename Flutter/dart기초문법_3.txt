< dart기초문법_3.txt>

1. 연산자
- +, -, *
- / : 나누기 (double형으로 반환)
- ~/ : 몫 (int형으로 반환)
- % : 나머지 (int형으로 반환)
※ C계열언어와 유사

2. 증감연산자
- 후위 연산 : i++, i--
- 전위 연산 : ++i, --i

3. 비교연산자 (결과는 bool형)
- == 
- !=
- >
- <
- >=
- <=
※ 연산 결과 : bool형

4. 논리연산자
- && (AND)
- ||  (OR)
- ==
- |
- !=

5. 비트연산자
- &  (AND)
- ^  (XOR)
- `  (OR)
- >>, <<, >>> (쉬프트)
