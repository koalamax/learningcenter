1. 개발 환경 구성
- Flutter SDK
- 다운로드: https://docs.flutter.dev/release/archive
- 설치: https://docs.flutter.dev/get-started/install/windows/desktop
- Android Studio
- 다운로드/설치: https://developer.android.com/studio?hl=ko


2. 개발 환경 구성 - flutter SDK (1/2)
- flutter_windows_3.xx.x.stable.zip 압축을 푼다.(예) flutter_windows_3.29.2-stable.zip
- 압축을 풀면 flutter란 디렉토리가 생성된다.
- 압축 푼 디렉토리를 C:\user\{로그인명}\dev\복사한다. 예) C:\Users\dimohy\dev\flutter


3. flutter/bin을 PATH 환경변수에 추가
- 시스템속성 -> 환경변수->Path를 편집
- %USERPROFILE$\dev\flutter\bin 추가

4. 터미널을 열어서 버전확인
- flutter --version을 쳐볼것
